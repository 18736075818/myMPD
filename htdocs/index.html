About

ympd - a lightweight and fast Music Player Daemon web client


ympd is written in pure C and based completely on HTML5 WebSockets with Bootstrap.
ympd uses following excellent software:
libWebSockets LGPL2.1 + static link exception
libMPDClient BSD License
